##其它
###别名

	git config --global alias.lg "log --color --graph --pretty=format:'%Cred%h%Creset -%C(yellow)%d%Creset %s %Cgreen(%cr) %C(bold blue)<%an>%Creset' --abbrev-commit"
运行上面语句后，然后命令 git lg 可以看到这样的效果![](/images/lg.jpg)

###git暂存
命令：
	
	#暂存当前状态
	git stash
	#列出暂存列表
	git stash list
	
	#恢复暂存
	git stash apply
	#删除暂存
	git stash drop
	#恢复并删除暂存
	git stash pop
